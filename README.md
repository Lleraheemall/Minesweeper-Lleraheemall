# Saper — Browser Minesweeper Game

A simple and modern browser version of the classic **Minesweeper** game built using **HTML**, **CSS**, and **JavaScript**.

## Overview

This project recreates the classic Minesweeper experience with a clean responsive interface and both light and dark themes.  
The game automatically opens an area on the first click and allows flag placement using the right mouse button.

## Features

- Three difficulty levels:
  - 8×8 — Beginner
  - 16×16 — Intermediate
  - 24×24 — Expert
- Light and dark theme toggle
- Timer
- Automatic area reveal after the first click
- Flag system (right-click)
- Restart after losing
- Responsive layout for desktop and mobile
